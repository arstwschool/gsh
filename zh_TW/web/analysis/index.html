分析
社會與自然組之區別
社會組在與自然組相比之下，自然組高達74.2%覺得午餐菜量不夠，而社會組只有11.5%。社會組在午餐各方面對於量的需求感到不足之比例明顯低於自然組。
這或許與班級組成之成員有相關聯，像是自然組男生較多因此食量較大，社會組則是相反。或許可以根據男女比例調整菜量而不單單只靠人數分配。
菜樣喜歡與否分析
從這兩份圖表可得知學生們最常吃完的是蛋類，接著是其他類，包含了甜筒、饅頭這類特殊食物。其次是肉類。可能是因為學生平均而言最喜歡蛋，肉類則是大部分喜歡。
學生喜愛菜色前五名中，討厭榜上有四項是菜類，也就是為什麼在圖四中菜類的剩菜量會最高了。
配合圖二，雖然喜歡蔬菜的比例與前兩名平分秋色，但也有不少人最討厭的是蔬菜，這或許男女之間的差異有關，又或是大家只喜歡特定的蔬菜。
而魚雖然是肉類，卻出現在討厭榜上第三名。可能是因為魚類常常有刺導致學生們不願意吃。

喜愛榜上高麗菜為第五名，往後或許可以多增加高麗菜的出現次數。
